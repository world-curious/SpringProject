# 环境
### Java 1.8

### Node.js 18.16.0+

### maven 2.6.3

# 下载运行代码

### 通过git
```sh
 git clone https://github.com/world-curious/SpringProject.git
```
### 或者直接下载
 点击code下载代码的压缩包(zip)

 下载解压完成后在IDE中打开

## 加载运行
首先加载maven项目，导入 pom.xml

找到 weblog-web 路径下的 WeblogWebApplication.java ，点击运行即可启动项目

找到 weblog-vue3 并在控制台中打开它

### 控制台运行下列指令下载依赖项
```sh
npm install i
```

### 控制台运行下列指令即可启动项目
```sh
npm run dev
```
## 关于报错：确保项目文件的编码为UTF-8
